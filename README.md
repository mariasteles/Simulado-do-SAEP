# 🚀 Gerenciador de Tarefas Kanban

**Nome:** Maria Eduarda Schmitt Teles

## 📌 Descrição

Este projeto consiste em uma aplicação de gerenciamento de tarefas no modelo Kanban (To-Do List), permitindo o controle de atividades em três etapas:

* A Fazer
* Fazendo
* Pronto

A aplicação permite o cadastro de usuários, criação de tarefas e gerenciamento visual das atividades.

## 🛠️ Tecnologias Utilizadas

* Node.js
* Express
* MySQL
* HTML, CSS e JavaScript

## 📂 Estrutura

* `/docs` → Diagramas DER e Caso de Uso
* `setup_banco.sql` → Script do banco
* `/backend` → Servidor
* `/frontend` → Telas do sistema

## ▶️ Como executar

1. Importar o arquivo `setup_banco.sql` no MySQL
2. Rodar o backend:

```
node server.js
```

3. Abrir os arquivos HTML no navegador
